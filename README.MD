GenAi-Research-Assistant
An AI-powered Research Paper Assistant built using Python, Transformers, FAISS, Sentence Transformers, and NLP. It enables semantic search, paper summarization, question answering, keyword extraction, named entity recognition, and document analysis through an interactive interface.

📚 GenAI Research Assistant
An AI-powered Research Paper Intelligence System that helps researchers efficiently analyze, search, summarize, and understand research papers using modern Natural Language Processing (NLP) and Generative AI techniques.

📄 PDF Research Paper Upload
🔍 Semantic Search using Sentence Transformers & FAISS

🔑 Keyword Extraction
🏷 Named Entity Recognition (NER)
📚 Research Paper Classification

📈 Interactive Visualizations
💻 User-friendly Interface
